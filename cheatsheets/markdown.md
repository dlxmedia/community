# HEADERS

### `📝 TO WRITE`
- ⚠️ Make sure you add a space after the # sign
- H1 and H2 add a line below automatically
- Add `#` or `---` to add a divider

```markdown
# H1 TITLE
## H2 TITLE
### H3 TITLE
#### H4 TITLE
```

### `👁️ WHAT YOU SEE`

# H1 TITLE
## H2 TITLE
### H3 TITLE
#### H4 TITLE

&nbsp;
### `🎨 STYLED HEADERS`

```markdown
H2 with Emoji + Highlight (code)
## `⚡️ Page Title`
```
 ## `⚡️ Page Title`

&nbsp;

```markdown
Quoted H3 with Line Below
```
