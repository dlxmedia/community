# HEADERS
> Make sure you add a `space` after the `#` sign
> 
> `H1 title` and `H2 title` add a line below automatically
> to force a line to be added, just type `#` below the header
> 

### `📝 TO WRITE`
```markdown
# H1 TITLE
## H2 TITLE
### H3 TITLE
#### H4 TITLE
```

### `👁️ WHAT YOU SEE`

# H1 TITLE
## H2 TITLE
### H3 TITLE
#### H4 TITLE

&nbsp;
### `🎨 STYLED HEADERS`

```markdown
H2 with Emoji + Highlight (code)

```
